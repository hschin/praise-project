# S03: Deck Editor — completed 2026-03-13

**Goal:** Users can assemble a complete service deck — adding songs, arranging slides, editing text, and choosing a visual theme — ready for export
**Demo:** User creates a deck, adds songs, reorders/edits slides inline, applies a visual theme, and previews the result

## Must-Haves

- User can create a service deck and add songs from the library
- User can reorder and remove songs without affecting the library
- User can edit, reorder, delete, and repeat individual slides inline
- User can apply a visual theme (AI-suggested, custom, or uploaded background)
