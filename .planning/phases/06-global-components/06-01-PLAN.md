  <action>
    Create app/javascript/controllers/flash_controller.js:

    ```javascript
    import { Controller } from "@hotwired/stimulus"

    export default class extends Controller {
      static values = {
        autoDismiss: { type: Boolean, default: false },
        delay:       { type: Number,  default: 4000 }
      }

      connect() {
        requestAnimationFrame(() => {
          this.element.classList.remove("translate-x-full", "opacity-0")
        })
        if (this.autoDismissValue) {
          this._timer = setTimeout(() => this.dismiss(), this.delayValue)
        }
      }

      dismiss() {
        clearTimeout(this._timer)
        this.element.classList.add("translate-x-full", "opacity-0")
        this.element.addEventListener("transitionend", () => this.element.remove(), { once: true })
      }

      disconnect() {
        clearTimeout(this._timer)
      }
    }
    ```

    Key points:
    - `requestAnimationFrame` is REQUIRED — without it the element renders already-visible and the CSS entry transition never fires
    - `data-flash-auto-dismiss-value="true"` triggers the 4s timer; false (default) means manual dismiss only
    - `dismiss()` transitions out then removes from DOM via `transitionend` (once: true prevents listener leak)

    Create app/views/shared/_flash_toast.html.erb:

    Accepts locals: `type` (`:notice` or `:alert`), `message` (string).
    - `is_success = type == :notice`
    - Success: `bg-green-50 border border-green-200`, `text-green-700` for text and icon, `data-flash-auto-dismiss-value="true"`
    - Error: `bg-red-50 border border-red-200`, `text-red-700` for text and icon, auto-dismiss stays false (default)
    - Both: `w-80 rounded-lg shadow-md px-4 py-3 flex items-start gap-3 pointer-events-auto`
    - Both: `transition-all duration-300 translate-x-full opacity-0` — initial hidden state; controller removes these on connect
    - `data-controller="flash"` on the toast div (NOT on the container — each toast has its own controller instance)
    - `data-flash-delay-value="4000"` on notice toasts
    - Include Heroicons inline SVG: check-circle for success, exclamation-triangle for error (use the exact SVG paths from the interfaces section above)
    - X button (x-mark SVG) only on error toasts: `class="text-stone-400 hover:text-stone-600 flex-shrink-0 ml-auto"` with `data-action="click->flash#dismiss"`
    - Message text: `<p class="text-sm flex-1 {text_class}"><%= message %></p>`
