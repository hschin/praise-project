## Actionable Issues Before Archiving

### 1. Fix ThemesController regression (THEME-01) — HIGH
**File:** `app/controllers/themes_controller.rb`
**Problem:** `attrs = theme_params.merge(source: "custom")` introduced in commit `9b2744f` reverts the fix from commit `ae25ffc`. AI-applied themes are saved as `source: "custom"`.
**Fix:** Change back to `@theme.source ||= "custom"` (or equivalent that doesn't overwrite params).

### 2. Commit uncommitted changes — HIGH
**Files:** `app/jobs/generate_theme_suggestions_job.rb`, `app/services/claude_theme_service.rb`
**Problem:** Working-tree changes (improved error messages, model name update to `claude-haiku-4-5-20251001`) are uncommitted and will be lost.
**Fix:** Review and commit these changes before archiving.

### 3. Phase 02 missing VERIFICATION.md — MEDIUM
**Problem:** gsd-verifier was never run for Phase 2. All 7 requirements are marked complete in REQUIREMENTS.md and human approval was recorded in 02-06-SUMMARY.md, but no formal VERIFICATION.md exists.
**Fix:** Run `/gsd:verify-work 2` or create a VERIFICATION.md for Phase 2 before archiving.

### 4. _failed.html.erb missing deck_id — LOW
**File:** `app/views/songs/_failed.html.erb`
**Problem:** Paste form after failed import doesn't carry `deck_id`. Song imported via paste-retry won't be auto-added to the deck.
**Fix:** Add `hidden_field_tag :deck_id, params[:deck_id]` to the form.
